Game Rules

Description

Crash is an online game where you can place stakes on a rapidly increasing multiplier. 

The objective of the game is to strategically cashout before the multiplier crashes. 

If you cashout before the multiplier crashes you receive a payout based on your original stake and the final multiplier.

If the multiplier crashes before you cashout, the stake is lost.

 

General Information

Game Type: Crash Game

The minimum stake is 20.00 ֏

The maximum stake is 50,000.00 ֏

Max Exposure: 50000000 ֏

RTP: 96%

Max Multiplier: 10,000x

 

How to play

 

Select the stake amount. 

Place the stake before the round starts.

Watch the multiplier increase.

Cashout at any time before the multiplier crashes.

 

Auto

The game features auto stake and cash-out functionality. Switch on Auto, and starting from the next round, stakes will be placed automatically. Switch off Auto during the round, the win will be cashed out immediately. 
For cashing out automatically, select your target multiplier and switch on the auto cashout.

Controls


“Start” buttons are used to place stakes. 

“-” “+” or suggested stake amounts can be used to change the stake amount.

"Sound" button allows you to change the in-game sound settings.

“History” opens your stake history.

"Information" button opens the game rules and controls section.

"Balance" label shows the current balance.



 

Game Interruptions

Game Recovery
If the game is interrupted due to a loss of connection, the next time the player accesses the game, the unfinished game will be automatically resumed at the state it was interrupted immediately before the interruption.

Cancelation

If the game has been canceled independently from the player, the moment of the cancelation is considered to be the end of the game. Any funds accumulated will be automatically returned to the player.

In the event of a malfunction, all stakes and wins are void.
